# pdf-color-changer

## About
This program splits a PDF into image slices, allows the user to replace specific colors in the PDF, and stitches the image slices back into an output PDF.

### Library    Versions
    img2pdf    0.5.1
    pdf2image  1.17.0
    PyPDF2     3.0.1
    pillow     10.2.0

#### Python Version: 3.11
